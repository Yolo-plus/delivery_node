## 实现功能
首页ECharts可视化展示，管理员、商家、商家食品增删改，用户详情，用户收货地址，用户搜索，分页实现，订单列表和详情，管理员登录，退出登录等

## 使用：
1. 导入数据库 - delivery.sql
2. 下载依赖 - npm install
3. 开启后台 - nodemon app.js
4. localhost:8000/index

## 注：
1. 登录信息位于adminlist表中
2. app.js第244行需要输入自己的阿里云AccessKey，否则上传文件的时候会报错
